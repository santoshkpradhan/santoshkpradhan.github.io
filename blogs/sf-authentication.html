Salesforce Authentication Mechanisms

by Santosh Pradhan on August 24, 2017

Anytime you need to integrate to Salesforce, you need to authenticate yourself first. There are options however to build public APIs (e.g., using Public Sites as an end-point). But you still need to set-up authentication for your Guest site user.

In this chapter, we would discuss and work on following ways of authenticating a user

  • Basic Authentication
  • OAuth 2.0
  • SOAP Login
  • OpenID Connect
